# FMB Power Prompter

Gerador e aprimorador de prompts para IA de vídeo, pronto para rodar no computador ou publicar online.

## Como rodar (passo a passo para leigos)

1. Instale o Node.js: [nodejs.org](https://nodejs.org)
2. Abra o terminal (Prompt de Comando no Windows; Terminal no Mac).
3. Entre na pasta do projeto:
   ```
   cd caminho/da/pasta/fmb-power-prompter
   ```
4. Instale as dependências:
   ```
   npm install
   ```
5. Rode o projeto:
   ```
   npm run dev
   ```
6. No navegador, acesse: http://localhost:3000

Pronto!

Se tiver dúvidas, peça ajuda por aqui.
